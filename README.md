# Хатинка Лапок 

Хатинка Лапок — це Сайт притулку для тварин. Мета проєкту — допомогти знайти нові домівки для безпритульних хвостиків, які потребують любові та піклування. На сайті можна переглянути тварин, що шукають родину (секція "Знайти друга"), дізнатися більше про притулок, ознайомитися з відгуками нових власників та знайти відповіді на часті запитання.

Проєкт побудований на базі **Vanilla JavaScript** з використанням збирача **Vite**.

## Технології

Додаток використовує наступні технології та бібліотеки:
- **HTML/CSS/JS** — базова розмітка, стилізація та логіка.
- **[Vite](https://vitejs.dev/)** — інструмент збирання та швидкий сервер для розробки.
- **[Axios](https://axios-http.com/)** — для здійснення HTTP-запитів до API (наприклад, для отримання списку тварин чи відгуків).
- **[Swiper](https://swiperjs.com/)** — для створення красивих слайдерів (секції "Про нас" та відгуків).
- **[IziToast](https://izitoast.marcelodolce.com/)** — для гарних вспливаючих сповіщень в разі помилки або успіху після відправки форми.
- **[Accordion JS](https://github.com/michu2k/Accordion)** — для реалізації секції FAQ.

## Як розгорнути проєкт локально

Щоб запустити проєкт на своєму комп'ютері, виконайте наступні кроки:

### 1. Клонування репозиторію
Склонуйте репозиторій на свій комп'ютер за допомогою Git:
```bash
git clone https://github.com/komandor131/project-PawFriends
cd project-PawFriends
```

### 2. Встановлення залежностей
Переконайтеся, що у вас встановлений [Node.js](https://nodejs.org/). Запустіть команду для встановлення всіх необхідних пакетів:
```bash
npm install
```

### 3. Запуск сервера для розробки
Щоб запустити проєкт у режимі розробки зі швидким оновленням (hot-reload), виконайте:
```bash
npm run dev
```
Після цього термінал покаже локальне посилання (зазвичай `http://localhost:5173/`), за яким можна переглянути сайт у браузері.

### 4. Збірка для продакшену
Щоб зібрати оптимізовану версію проєкту, виконайте:
```bash
npm run build
```
Зібрані файли з'являться у папці `dist/`.

Для попереднього перегляду зібраної версії локально:
```bash
npm run preview
```

## Структура проєкту

- `/src` — вихідні файли проєкту.
  - `/partials` — окремі компоненти HTML (хедер, футер, секції).
  - `/css` — стилі для кожної секції.
  - `/js` — логіка кожного компонента.
  - `/img` — зображення, іконки та SVG спрайт.
- `package.json` — конфігураційний файл пакета з залежностями та скріптами.
- `vite.config.js`— налаштування збирача.
